# Phishing URL Detection using Machine Learning

This project detects whether a URL is phishing or legitimate using machine learning classification algorithms.

## 📌 Project Overview
Phishing attacks trick users into entering sensitive information on fake websites.  
This project builds a machine learning model to classify URLs as phishing or legitimate.

## 🧠 Algorithms Used
- Logistic Regression
- Random Forest

## ⚙️ Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## 📂 Project Files
- minor_1.ipynb → Model training and evaluation
- requirements.txt → Required Python libraries

## 🚀 How to Run
1. Install dependencies  
pip install -r requirements.txt

2. Open the notebook and run all cells.

## 📊 Output
The model predicts whether a given URL is phishing or safe.
